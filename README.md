# wii-catch
Catch Game is a Wii-inspired interactive game built using p5.js and WebGL.
Players catch incoming fruit using a motion-style cursor, earning combo bonuses and avoiding misses. The project features a custom scene/state system, animated UI, 3D projectile mechanics, particle effects, and layered 2D/3D rendering.

Originally developed as part of an A-Level Computer Science NEA project, with emphasis on modular design, object-oriented architecture, and iterative development.
